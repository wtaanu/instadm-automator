# Instagram Automation Client Setup Guide

## Purpose
This document explains:

1. what the platform does for your Instagram account
2. what you need to prepare before setup
3. how your team will use the dashboard
4. what is automated and what still needs approval

## What This System Does
The platform helps your team manage Instagram growth in a more organized way.

It is designed to:

- track account performance
- organize content planning
- monitor incoming comments and DMs
- detect intent such as question, inquiry, or collaboration
- trigger reply workflows and DM follow-up paths
- show progress through analytics and activity reports

This is not a spam bot. It is a workflow and growth operations system.

## What You Need Before Setup
Please keep the following ready before onboarding begins.

### 1. Instagram Account Requirements
- A professional Instagram account
- Business or Creator profile enabled
- Admin access to the connected Instagram account
- The Instagram account connected to a Facebook Page

### 2. Meta Access Requirements
- Access to the Meta Business / Facebook account connected to Instagram
- Permission to approve Meta login during the connection step
- Permission to the Facebook Page connected to the Instagram account

### 3. Business Information
- Brand name
- Niche / industry
- Primary business goal
  Examples:
  - increase reach
  - generate leads
  - grow followers
  - improve engagement
- Preferred posting frequency
- Team size or number of people managing Instagram

### 4. Destination Links
Please provide the links you want the system to use inside DM workflows.

- Sales link
  Example: book a demo / contact sales / pricing page
- Course link
  Example: training page / course checkout / workshop page
- Community link
  Example: WhatsApp group / Discord / private community / newsletter

### 5. Brand Messaging Inputs
- Short description of your offer
- Common FAQs
- Preferred tone of communication
- If available, sample captions or existing messaging guidelines

## Pre-Setup Checklist
Before going live, the client should confirm:

- Instagram login is available
- Meta login is available
- Facebook Page is connected correctly
- Sales, course, and community links are final
- Brand tone and approved response style are confirmed
- Team knows who will review high-intent conversations

## Setup Flow

### Step 1. Workspace Setup
During onboarding, we add:

- brand name
- niche
- goal
- posting frequency
- team size
- account access status
- sales, course, and community links

### Step 2. Instagram Connection
We connect your Instagram account using Meta login.

This connection allows the system to:

- identify the connected Instagram account
- store the account reference securely
- sync media, comments, and available insight data

### Step 3. Comment and DM Routing Setup
We configure routing rules such as:

- question -> course link
- inquiry -> sales link
- collaboration -> community or partner path

These rules can be edited later from the dashboard.

### Step 4. Sync and Review
After setup, we run the first sync to pull:

- profile details
- media
- comments
- available analytics data

Then we review:

- what data is available
- what workflows are active
- what still requires manual approval

## How the Client Uses the Dashboard

### Overview
Use this section to understand:

- account health
- recent growth progress
- performance direction
- top priorities for the next actions

### Content
Use this section to:

- review the content calendar
- see trend signals
- generate new hooks and CTAs
- copy hooks for the content team

### Inbox
Use this section to:

- review high-priority DMs
- review comment queue
- follow recommended next actions
- reduce reply delays

### Analytics
Use this section to:

- review performance snapshots
- monitor automation outcomes
- track message opens and clicks
- see how intent routing is performing

### Settings
Use this section to:

- update brand information
- update destination links
- connect or sync Instagram
- adjust auto-DM rules

## What Is Automated
Depending on setup and account permissions, the system can help automate:

- comment listening
- intent detection
- DM routing logic
- content idea generation
- hook and CTA generation
- job scheduling for sync tasks

## What Still Needs Human Review
The following should usually remain reviewed by a person:

- pricing conversations
- partnership conversations
- custom client requests
- unusual or sensitive replies
- major brand messaging changes

## Recommended Client Workflow

### Daily
- check Inbox
- reply to high-priority leads
- review new comments

### Weekly
- review Overview and Analytics
- approve next content ideas
- generate new hooks and CTAs
- check if destination links or reply logic need updates

### Monthly
- review growth trends
- review which workflows are converting best
- refine routing rules and CTA strategy

## Important Notes
- Some Instagram insight data depends on what Meta makes available for the connected account
- If insight access is limited, the dashboard may still show media and workflow data even when some analytics are unavailable
- The system works best when the account is properly connected through Meta and reviewed regularly by the team

## Support Information To Keep Ready
If troubleshooting is needed, please keep these ready:

- Instagram handle
- connected Facebook Page name
- Meta admin email
- screenshot of the issue
- what action was being performed when the issue happened

## Final Recommendation
For best results, clients should treat this system as a growth operations dashboard, not just a reporting screen.

The strongest outcomes come when the team:

- responds quickly
- keeps links updated
- uses generated hooks regularly
- reviews insights weekly
- refines routing based on actual conversations
